# FastAPI Training Project 🚀

Учебный проект по созданию API для магазина/каталога товаров с использованием **FastAPI** и **SQLAlchemy**.

## 🛠 Стек технологий
* **Python 3.10+**
* **FastAPI** — современный веб-фреймворк.
* **SQLAlchemy** — ORM для работы с базой данных.
* **SQLite** — база данных (хранится в локальном файле).
* **Pydantic** — валидация данных.
* **Uvicorn** — сервер для запуска приложения.

## 📋 Функционал
- [x] Работа с пользователями (создание, получение, обновление, удаление).
- [x] Работа с товарами (Items) и привязка их к владельцу (User).
- [x] Автоматическая интерактивная документация (Swagger UI).
- [x] Фильтрация товаров по названию и цене.

## 🚀 Как запустить проект

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com
   cd FastAPI-train
   ```

2. **Создайте и активируйте виртуальное окружение:**
   ```bash
   python -m venv .venv
   # Для Windows:
   .venv\Scripts\activate
   ```

3. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Запустите сервер:**
   ```bash
   python main.py
   ```
   После запуска API будет доступно по адресу: `http://127.0.0.1:8000`

## 📖 Документация
После запуска сервера перейдите по адресам:
* **Swagger UI:** `http://127.0.0` — здесь можно тестировать все запросы прямо в браузере.
* **ReDoc:** `http://127.0.0`

## 🗄 Структура проекта
* `main.py` — эндпоинты (роуты) и логика API.
* `db.py` — конфигурация базы данных и SQLAlchemy модели.
* `test.db` — локальная БД SQLite (не коммитится).
